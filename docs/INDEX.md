# INDEX.md — Project Overview

## What is this project?

A Tetris game built for mobile platforms, starting with Android and later expanding to iOS (Apple).

## Goals

- Deliver a fully functional Tetris experience on Android
- Port to iOS after the Android version is stable
- Keep the codebase clean and well-documented for long-term maintenance

## Q&A

**Why Tetris?**
Classic game, well-defined rules, great scope for learning mobile game development.

**What platforms are targeted?**
- Android (primary)
- iOS (planned after Android release)

**Is there a web version planned?**
Not currently. Focus is on native mobile.
